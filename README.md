# Pong Game

A basic pong game using SVGs. The technologies used in this game are: html, css and javascript

## Setup
download game
unzip all files into a folder

**Install:**

`> npm i`

**Run:**

`> npm start`

## Usage

Open index.html

**Player 1: - on the left handside **
* a: up
* z: down

**Player 2: - on the right handside**
* ▲ : up
* ▼: down

**Pause: - spacebar key **
